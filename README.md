# windrives

> Because Microsoft decided that getting basic drive info requires calling 4 different functions with names that sound like they were generated by a random WinAPI name generator.

So instead of writing this every time:

```c
GetLogicalDrives();
GetDriveTypeW(path);
GetDiskFreeSpaceExW(path, ...);
GetVolumeInformationW(path, ...);
```

You write this:

```c
GetDrives(drives, 26);
```

You're welcome.

---

## Installation

No installation. Drop `windrives.h` and `windrives.c` into your project and compile.

```sh
cl your_code.c windrives.c
# or
gcc your_code.c windrives.c -o your_program
```

---

## Why does this exist?

At some point, someone at Microsoft looked at a drive and thought:
*"You know what, the letter, label, filesystem, serial number, free space and type — these should each require a separate function call with different argument conventions."*

This is a silent protest against that decision.

## API

### Structures

```c
typedef struct {
    WCHAR  cLetter[4];          // Drive letter, e.g. L"C:\"
    WCHAR  szLabel[MAX_PATH];   // Volume label, e.g. L"Local Disk"
    WCHAR  szFileSystem[MAX_PATH]; // e.g. L"NTFS", L"FAT32"
    DWORD  uType;               // DRIVE_FIXED, DRIVE_REMOVABLE, etc.
    DWORD  dwSerialNumber;      // Volume serial number
    DWORD  dwMaxComponentLen;   // Max filename length
    DWORD  dwFileSystemFlags;   // Filesystem feature flags
    double dTotalGb;            // Total size in gigabytes
    double dFreeGb;             // Free space in gigabytes
} DRIVEINFO, *LPDRIVEINFO;
```

### Functions

```c
// Get all drives (except DRIVE_UNKNOWN and DRIVE_NO_ROOT_DIR).
// Returns number of drives found.
UINT GetDrives(LPDRIVEINFO lpDst, UINT uCount);

// Same, but only removable drives (USB sticks, SD cards, etc.)
UINT GetRemovableDrives(LPDRIVEINFO lpDst, UINT uCount);

// Just the count, no info.
UINT GetDrivesCount();

// Get info for a specific drive letter.
// Returns FALSE if drive doesn't exist or lpDst is NULL.
BOOL GetDriveByLetter(WCHAR letter, LPDRIVEINFO lpDst);

// Block until a new drive is connected or timeout expires.
// Pass INFINITE if you enjoy waiting forever.
void WaitForNewDriveConnected(DWORD dwTimeoutMs);
```

## Usage

```c
// Print all drives
DRIVEINFO drives[26];
UINT count = GetDrives(drives, 26);

for (UINT i = 0; i < count; i++) {
    wprintf(L"%s  %.1f GB free / %.1f GB total\n",
        drives[i].cLetter,
        drives[i].dFreeGb,
        drives[i].dTotalGb);
}

// Wait for a USB drive to be plugged in (up to 10 seconds)
WaitForNewDriveConnected(10000);

// Get info on C: specifically
DRIVEINFO c;
if (GetDriveByLetter(L'C', &c))
    wprintf(L"Label: %s\n", c.szLabel);
```
